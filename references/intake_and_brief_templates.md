# Intake and Brief Templates

Use these templates when the user wants a structured intake form, a handoff brief, or a reusable article planning document.

## Draft Intake

```markdown
## 草稿
[粘贴草稿]

## 我想发布的平台
微信公众号 / Medium / 小红书长文 / Newsletter / Blog / 其他：

## 目标读者
[谁会看？他们现在有什么困惑？]

## 我希望读者带走什么
[一个判断 / 一个方法 / 一个清单 / 一个故事]

## 我希望文章的感觉
专业 / 真诚 / 锋利 / 温和 / 有传播感 / 有干货 / 其他：

## 不能虚构或不能公开的信息
[客户名、数据、项目细节、真实身份等]
```

## Article Brief

```markdown
## Article Brief

### Platform

### Target reader

### Reader pain

### Core promise
这篇文章帮助 [目标读者] 通过 [作者经验] 理解/掌握/避免 [核心问题]。

### Core thesis

### Personal story material

### Method material

### Evidence / details

### Boundaries

### Recommended structure

### Desired output
```
